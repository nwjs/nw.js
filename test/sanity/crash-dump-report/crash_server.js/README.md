# a simple nwjs crash server of nodejs

## Test step

```sh
npm i -g @licq/tnwjs
tnw install 0.67.1 # or 0.57.1


npm i
# start server
npm run server
# start nwjs
npm run app
```

## Clear the NW.js processes

> When NW.js crashed, for some old NW.js version(eg: 0.57.1), you should manually clear the NW.js processes

```sh
npm run clear
```
